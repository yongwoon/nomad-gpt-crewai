## Getting start with venv

- set venv

```bash
# foramt: python -m venv [FOLDER NAME]
python -m venv ./crew-ai
```

- activate env

```bash
source crew-ai/bin/activate
```

- deactivate

```bash
deactivate
```

- install package

```bash
pip install --upgrade pip

pip install -r requirements.txt
```
